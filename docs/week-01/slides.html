Machine Learning

It can be really hard to explicitly programme the computer to solve tasks with real-life complexity.

Machine learning is a subfield of artificial intelligence that gives computers the ability to learn without explicitly being programmed. It learns from data.

Machine learning model is the product of machine learning, which takes some input, does some process and makes outputs.

Input -> Process -> Output

(we will refer back to this multiple times throughout this unit)
